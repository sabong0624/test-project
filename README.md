## 코드 넣는법
### 파이썬에서 Hello World!
```python
name = input('이름을 입력하시오: ')
print(name, '님 환영합니다!')
```

### Java에서 Hello World!
```Java
public class HelloWorld {
	public static void main(String[] args) {
		System.out.println("Hello World!");
	}
}
```

문장 사이에 어떤 코드를 강조 할때, 변수 `name`에는 이름이 저장된다.
